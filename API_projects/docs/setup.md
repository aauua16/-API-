## スプレッドシートの列定義の書き方

- 列・項目名・説明
- A列・ISBN・検索したい本の13桁ISBNを入力（ハイフン有無は自動調整）
- B列・調布ステータス・「蔵書あり」「なし」「確認中...」が自動入力される
- C列・調布詳細・蔵書場所と状態（貸出中など）が自動入力される
- D列・調布URL・予約ページへのリンクが自動入力される
- ・・・
- ※注意: スクリプト内の getRange 関数で列番号を指定しているため、列を入れ替えたり削除したりすると正しく動作しません。

## APIキー（スクリプトプロパティ）の設定手順
APIキーを安全に利用するため、以下の手順で設定を行ってください。
1. スプレッドシートのメニューから 「拡張機能」 > 「Apps Script」 を開く。
2. 左側のサイドバーにある 「設定（歯車アイコン）」 をクリックする。
3. 画面下部の 「スクリプトプロパティ」 セクションにある 「スクリプトプロパティを編集」 をクリックする。
4. 以下の内容でプロパティを追加する：
  5. プロパティ: CALIL_API_KEY
  6. 値: あなたがカーリルから取得したAPIキー
7. 「スクリプトプロパティを保存」 をクリックして完了。
